Powerful Python

May 8, 2021   

Python is a tool that is similar and as useful as R Studio. Python can be used as a calculator, can manipulate strings, can create lists, and can be used as many other functions. Additionally, Python can create plots. Below are some examples:

# python as a calculator
3-2
## 1
3+2
## 5
3**2
## 9
# indexing and lists
list1=[1,2,3,4]
list2=['a','b','c','d']
list1[0]
## 1
list2[1]
## 'b'
list2[2:]
## ['c', 'd']
# reticulate

library(reticulate)

hi<-"Hello"
hi="world"
print(r.hi,hi)
## Hello world

Through reticulate, we are now able to allow R and Python to play together. Objects can be named the same thing in R and Python without conflict.
